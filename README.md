# Portofolio Mahasiswa Informatika - Universitas Hamzanwadi

Website portofolio modern dan responsif untuk mahasiswa informatika.

## Cara Menggunakan
1.  **Edit Isi**: Buka file `index.html` dan ganti teks placeholder (seperti "Nama Kamu", daftar projek) dengan data asli Anda.
2.  **Ganti Foto**: Ganti gambar placeholder dengan foto profil asli Anda dan screenshot projek Anda.
3.  **Deploy ke GitHub**:
    -   Buat repository baru di GitHub.
    -   Upload file-file ini (`index.html`, `style.css`, `script.js`).
    -   Masuk ke Settings > Pages pada repository GitHub Anda.
    -   Pilih Branch `main` dan folder `/root`, lalu Save.
    -   Website Anda akan online dan bisa diakses orang lain!

## Fitur
-   Desain Modern (Dark Mode + Glassmorphism)
-   Responsif (Tampilan menyesuaikan HP dan Laptop)
-   Animasi Scroll halus
